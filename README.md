De Olho No PIX

Nos somos uma ferramenta para ajudar usuários a verificar se uma chave PIX já recebeu denúncias de golpe. Funciona como um “Reclame Aqui” para PIX: antes de enviar seu dinheiro, você pode checar se alguém já reportou problemas com essa determinada chave.

Passo a passo de como usar:

1. Acesse o site.
2. Crie sua conta.
3. Consulte uma chave PIX ou cadastre uma denúncia.
4. Veja se o PIX já possui registros de fraude.
